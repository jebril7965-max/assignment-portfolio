Alex Rivera

I'm a software engineer and researcher at Microsoft Research in Seattle, where I work on machine learning systems and scalable distributed computing.

I completed my PhD at University of Washington, advised by Prof. Sarah Chen. My work has been recognized with the ACM SIGSOFT Distinguished Paper Award. Before grad school, I studied Computer Science at MIT and interned at Meta AI and Apple.

Projects

I'm interested in distributed systems, machine learning infrastructure, and developer tooling. My recent work focuses on making large-scale ML training faster and more reliable. Highlighted projects are ones I'm most proud of.

StreamTrain: Fault-Tolerant Distributed ML Training at Scale
Alex Rivera, Jamie Patel, Mei Lin, Carlos Ortega
OSDI, 2024   (Best Paper Award)
GitHub / arXiv / video

A distributed training framework that automatically recovers from node failures mid-training, reducing wasted compute by up to 60% on large GPU clusters.

AutoDebug: LLM-Powered Automated Bug Localization in Production Systems
Alex Rivera, Priya Nair, Tom Weston
ICSE, 2024
GitHub / arXiv

Uses large language models to automatically pinpoint the root cause of production bugs from logs and stack traces, cutting mean time to resolution by 45%.

NeuralCache: Learned Caching Policies for Key-Value Stores
Alex Rivera, Yuki Tanaka, Fatima Al-Hassan
SOSP, 2023
GitHub / paper

A reinforcement learning approach to cache eviction that outperforms LRU and LFU by 30% on real-world workloads from cloud storage systems.

CodeLens: Real-Time Code Quality Analysis in CI/CD Pipelines
Alex Rivera, Marcus Johnson
FSE, 2023
GitHub / paper

A lightweight static analysis tool that integrates into CI/CD pipelines and flags performance regressions and security vulnerabilities before they reach production.

GreenML: Carbon-Aware Scheduling for ML Workloads
Alex Rivera, Aisha Okonkwo, Lena Müller
EuroSys, 2022
GitHub / arXiv

A scheduler that shifts ML training jobs to times and regions with lower carbon intensity, reducing the carbon footprint of model training by up to 40%.
